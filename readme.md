# Computer from NAND

Building a computer from first principles.

## Goals

- Complete nand2tetris
- Reimplement all hardware in Verilog
- Synthesize the design for Tang Nano 9K
- Build supporting software in C
- Design a tiny neural-network accelerator (NPU)

## Repository structure

- nand2tetris/ — original course solutions
- verilog/ — synthesizable Verilog implementation
- tangnano9k/ — FPGA project
- software/ — tools and emulators
- docs/ — notes and architecture
